# Proper Way For Api Integration

Demo project for [https://pyaesoneaung.dev/proper-way-for-api-integration](https://pyaesoneaung.dev/proper-way-for-api-integration)

# Installation
```bash
composer install
cp .env.example .env
php artisan key:generate
```

